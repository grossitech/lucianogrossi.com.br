## Sobre

Site estático hospedado via **GitHub Pages**, sem framework, sem bundler, sem dependências — HTML, CSS e JavaScript puros.

## Páginas

| Rota | Descrição |
|------|-----------|
| `/` | Homepage / portfólio |
| `/gerador-link-whatsapp/` | Ferramenta: gerador de link direto para WhatsApp |
| `/contador-caracteres/` | Ferramenta: contador de caracteres por rede social |
| `/venca-os-mitos-que-sabotam-sua-carreira/` | Landing page do livro |

## Stack

- HTML5 · CSS3 · JavaScript (vanilla)
- Fontes: [DM Sans](https://fonts.google.com/specimen/DM+Sans) e [DM Mono](https://fonts.google.com/specimen/DM+Mono) via Google Fonts
- Hospedagem: GitHub Pages
- Domínio: Registro.br com DNS apontando para GitHub Pages

## Deploy

O deploy é automático a cada `git push` na branch `main`.

```bash
git add .
git commit -m "sua mensagem"
git push
```
